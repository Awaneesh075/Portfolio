# Avi Portfolio

This is a Next.js portfolio app (App Router) located under `src/`.

Quick start:

- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`
- Lint: `npm run lint`
- Type check: `npm run type-check`

Notes:
- Add your resume to `public/resume/Avi.pdf` (used by `Resume` component).
- Add project images to `public/projects/` as referenced in `src/data/projects.ts`.
